Planning

This is the planning page of the system. By pressing 'Create planning' you can generate a new planning. Creating a planning brings all the other elements together. Creating a planning or schedule is done via the following step-by-step plan:
  1. Make sure that all information from other pages are correct, i.e., the availabilities are correct, the worker skills are correct, etc.
  2. Create a planning by selecting the appropriate date. Then determine the minimum and maximum outputs per skill. This is largely pre-filled based on the outstanding tasks. However, it is not necessary to use the tasks function in this system.
  3. Press submit. The algorithm will determine the optimal schedules based on four objective functions:
    • Output maximization
    • Gross profit maximization
    • Preference maximization
    • Skill progression maximization
  4. A spider plot shows the trade-offs between the several objective functions. In the corresponding table, the values are shown. Furthermore, for every objective function, the optimal schedule is given, based on who is most suitable for which skill during which shift.
